변수와 자료형

변수 선언 시 var, let, const라는 키워드를 사용해서 선언

* 변수 선언 방법

  • var: 중복 선언 가능(단점), 재할당 가능
  • let: 중복 선언 불가, 재할당 가능
  • const: 중복 선언 불가, 재할당 불가

* 자료형

변수 선언 시 자료형을 별도로 지정하지 않음 (자료형의 개념이 없는 것은 아님)
=> 한 줄씩 실행되는 인터프리터 언어 특성상 변수에 대입되는 값에 따라서 자료형을 결정

자바스크립트의 자료형

  • 숫자: Number
  • 문자열: String
  • 논리 값: Boolean
  • 초기화 되지 않은 데이터: undefined
  • 값이 없는 데이터: null
  • 객체/배열/함수: Object

* 변수 명명 규칙

  1. 일반적으로 영어(소문자, 대문자)를 사용하고, 문자와 숫자를 모두 사용 가능
  2. 특수 문자 `$`, `_` 만 사용 가능
  3. 숫자로 시작할 수 없음
  4. 키워드(예약어)를 변수명으로 사용할 수 없음

표기법 (Naming Type)

  1. camelCase: 자주 사용되는 표기법
    소문자로 시작하고 다른 단어가 시작될 때마다 대문자로 구분
    • user id - userId
    • product name - productName
  2. snake_case: 파이썬에서 주로 사용되는 표기법.
    단어마다 `_`로 구분
    • user id - user_id
    • product name - product_name
  3. PascalCase: c# 또는 ms 계열의 언어에서 사용되는 표기법
    모든 단어의 시작을 대문자로 구분
    • user id - UserId
    • product name - ProductName

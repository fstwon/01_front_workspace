연습 4) 생성자 함수 만들기

HINT

속성

  • value: 현재 계산기의 값을 나타내며 초기값은 0입니다.

메서드

  • add(number): value에 number를 더합니다.
  • subtract(number): value에서 number를 뺍니다.
  • reset(): value를 0으로 초기화합니다.
  • getValue(): 현재 value를 반환합니다.
CODE 
              function Calculator() {
                // 여기에 코드를 작성하세요.
              }

              const calc = new Calculator();

              calc.add(10);
              console.log(calc.getValue()); // 출력: 10

              calc.subtract(4);
              console.log(calc.getValue()); // 출력: 6

              calc.reset();
              console.log(calc.getValue()); // 출력: 0

jQuery

* jQuery란?

기존 javaScript 언어만으로 구현하기 복잡했던 구문을 간소화하기 위해 만들어진 JavaScript 기반의 라이브러리 (유용한 기능 모음집)
즉, JavaScript 언어로 유용한 함수나 내용들을 다른 누군가 정의해놓은 것
복잡한 JavaScript 코드보다 짧고 쉽게 개발 가능

장점

  1. DOM 요소와 관련된 스크립트를 쉽게 구현할 수 있음
  2. AJAX(비동기 통신 방법), 이벤트 처리 등의 기능을 폭넓게 지원
  3. jQuery 관련 확장 플러그인, 오픈 API 등 지원(chart, slide, calendar, ...)

* jQuery 연결 방법

head 태그 내 script 태그로 외부 jQuery 관련 .js 파일 연결 (온라인 / 오프라인 방식)
- jQuery 공식 사이트: jQuery 사이트

  1. 온라인 방식(CDN): 파일을 직접 다운 받지 않고 html 문서를 파싱하면서 외부 url을 통해 script로 연결하여 받는 방식
  2. 오프라인 방식: jQuery 파일을 직접 다운로드 받아 script에 직접 연결하는 방식
    • Compressed(배포전용): .min.js
      주석, 들여쓰기, 띄어쓰기 등을 최소화하여 파일의 용량을 줄여 제공되는 파일
    • UnCompressed(개발자전용): .js
      개발자가 보기 쉽게 주석, 코드 정렬이 되어 있는 파일 (가독성 향상)

* jQuery 구문 작성 방법

jQuery 구문으로 DOM 요소를 다루는 구문 사용을 위해
문서 상 모든 요소들이 만들어진 이 후 실행
=> 순수 자바스크립트에서는 window 객체 onload 속성 내 작성 

      [window.]onload = function() {
        // 화면이 모두 렌더된 후 실행할 코드
      }
      * window.onload의 경우 해당 문서에서 한번만 실행


=> jQuery의 onload 구문 (3가지) 
      (1) jQuery(document).ready(function() {
            // 실행 코드
          });

      (2) $(document).ready(function() {
            // 실행 코드
          });

      (3) $(function() {
            // 실행 코드
          });
      * 제한 없이 여러 번 사용 가능. 순서대로 실행
